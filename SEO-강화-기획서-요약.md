## 🔧 기술적 SEO 구현

### 1. 메타 태그 최적화
```typescript
// 모든 페이지에 적용
export const metadata: Metadata = {
  title: '최적화된 Title (60자 이내)',
  description: '최적화된 Description (120~160자)',
  alternates: {
    canonical: 'https://pola.com/현재-경로',
  },
  openGraph: {
    title: 'OG Title',
    description: 'OG Description',
    images: [{ url: '/og-image.jpg', width: 1200, height: 630 }],
  },
};
```

### 2. Structured Data (JSON-LD)

**Organization Schema** (전역):
```json
{
  "@type": "Organization",
  "name": "POLA 정책자금 컨설팅",
  "url": "https://pola.com"
}
```

**Service Schema** (서비스 페이지):
```json
{
  "@type": "Service",
  "serviceType": "정책자금 컨설팅"
}
```

**FAQ Schema** (FAQ 페이지):
```json
{
  "@type": "FAQPage",
  "mainEntity": [...]
}
```

**Article Schema** (블로그):
```json
{
  "@type": "Article",
  "headline": "블로그 제목"
}
```

### 3. Sitemap & Robots.txt

**Sitemap.xml**:
- 모든 페이지 자동 포함
- 블로그 글 동적 추가
- Google Search Console에 제출

**Robots.txt**:
```
User-agent: *
Allow: /
Disallow: /api/
Sitemap: https://pola.com/sitemap.xml
```

### 4. 이미지 최적화
- WebP 포맷 사용
- alt 텍스트 필수 (키워드 포함)
- lazy loading 적용
- Next.js Image 컴포넌트 사용

### 5. 속도 최적화
- Lighthouse 성능 90점 이상
- LCP < 2.5초
- FID < 100ms
- CLS < 0.1
