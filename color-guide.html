Usage Guide

컬러 사용 가이드

헤더 / 네비게이션

배경: Primary Gradient, 텍스트: White, CTA: Accent (#3B82F6)

CTA 버튼 / 강조

배경: Accent (#3B82F6), hover 시 Accent Dark (#2563EB)

히어로 섹션

배경: Hero Gradient (#1D4ED8 → #1E40AF), CTA: Accent

푸터

배경: Gray 900 (#111827), 텍스트: Gray 400 (#9CA3AF)

카드 / 섹션

배경: White 또는 Gray 50 (#F9FAFB), 보더: Gray 200

링크 / 인터랙티브

Primary (#1D4ED8), hover 시 Primary Dark (#1E40AF)

CSS Variables

복사하여 사용할 수 있는 CSS 변수

:root {
  /* Primary Colors - 딥 네이비 블루 (정부/관공서 스타일) */
  --primary: #1D4ED8;
  --primary-light: #2563EB;
  --primary-dark: #1E40AF;
  --primary-pale: #DBEAFE;

  /* Accent Colors - 브라이트 블루 (CTA, 강조) */
  --accent: #3B82F6;
  --accent-light: #60A5FA;
  --accent-dark: #2563EB;
  --accent-pale: #EFF6FF;

  /* Status Colors */
  --success: #10B981;
  --warning: #F59E0B;
  --error: #EF4444;
  --info: #3B82F6;

  /* Neutral Colors */
  --neutral-50: #F9FAFB;
  --neutral-100: #F3F4F6;
  --neutral-200: #E5E7EB;
  --neutral-300: #D1D5DB;
  --neutral-400: #9CA3AF;
  --neutral-500: #6B7280;
  --neutral-600: #4B5563;
  --neutral-700: #374151;
  --neutral-800: #1F2937;
  --neutral-900: #111827;

  /* Gradients */
  --gradient-primary: linear-gradient(135deg, #1D4ED8 0%, #2563EB 100%);
  --gradient-hero: linear-gradient(135deg, #1D4ED8 0%, #1E40AF 100%);
  --gradient-accent: linear-gradient(135deg, #3B82F6 0%, #60A5FA 100%);
}
